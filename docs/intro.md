# Baleia

A Baleia é um protocolo de comunicação, settlement e reputação para sistemas multi-agent. Esta documentação consolida a tese do projeto, a arquitetura técnica, o fit com a Solana e o roadmap de execução definido para o contexto do **Colosseum Frontier**.

## Leitura recomendada

Para entender a tese do projeto de forma progressiva, a sequência recomendada é:

1. problema e solução
2. fit com a Solana
3. arquitetura técnica
4. mercado e modelo de negócio
5. diferencial competitivo
6. roadmap e time

## Objetivo do site

O objetivo deste site é servir como base pública de documentação da Baleia no GitHub Pages, reunindo em um único lugar:

- a narrativa do produto
- a justificativa técnica da escolha da Solana
- a arquitetura de referência do sistema
- o contexto competitivo e de mercado
- os materiais de suporte para pitch e submissão
